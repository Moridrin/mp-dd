# MP D&D
With this plugin you can write everything you want to share with the players based on a rights system. This way, you can write a page for a player and specify viewing rights for specific parts of it.
This is a plugin includes a wide variety of tags to generate links, modals, etc. Furthermore, the plugin allows you to tag game elements and toggle both game elements and all colors (including link colors) to give the reader a cleaner read.

## Installation
1. Upload the plugin files to the `/wp-content/plugins/mp-dd` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the SSV Options->Frontend Members screen to configure the plugin
