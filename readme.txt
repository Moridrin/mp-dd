With MP D&D you have lots of functionality to keep track of your D&D world.

== Description ==

MP D&D is a plugin that allows you to manage objects and events the way you want. With this plugin you can:
 - Manage all objects (these can be Characters, Places, Items, you name it).
 - Manage all events that have happened.
 - Link those events to the correct characters, places, etc.
 - Etc.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mp-dd` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
